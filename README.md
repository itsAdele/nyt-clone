New York Times Clone 📰
Progetto realizzato come parte del mio percorso di studi per mettere in pratica le competenze su React.

Descrizione
Un'applicazione web che replica le funzionalità principali del sito del New York Times. L'app permette di consultare le Top Stories divise per categoria e di effettuare ricerche mirate su specifici argomenti tramite l'API ufficiale del New York Times.

Tecnologie Utilizzate
React: Framework principale.

Axios: Per la gestione delle chiamate API asincrone.

React Router: Per la navigazione tra le pagine (Home e Best Sellers).

React Hooks: Utilizzo di useState e useEffect per la gestione dinamica dello stato.

CSS: Per un design responsive e intuitivo.

Funzionalità principali
Consultazione delle notizie in tempo reale suddivise per categorie (World, Politics, Tech, ecc.).

Barra di ricerca dinamica per trovare articoli specifici.

Pagina dedicata ai Best Sellers (libri).

Gestione degli errori di rete e visualizzazione dinamica dei contenuti multimediali.

Come eseguire il progetto
Clona questo repository.

Esegui npm install per installare le dipendenze.

Esegui npm start per avviare l'applicazione in locale.
